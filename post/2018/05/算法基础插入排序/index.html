算法基础———插入排序

2018年5月7日
| 算法基础

概念

  • 在归并排序中，我们是用一种被称为分治法的通用模式，在分治法中，我们即将原问题分解为类似原问题的子问题，并递归的求解这些子问题，然后再归并这些子问题的解。

复杂度

  • 时间复杂度 对于所有情况，它仅有一个O(nlgn)
  • 空间复杂度O（n）

原址

  • 非原址

js程序代码

bash function insertionSort(array) { 　　console.time('插入排序耗时：'); 　　for (var i = 1; i < array.length; i++) { 　　　　var key = array[i]; 　　　　var j = i - 1; 　　　　while ( array[j] > key) { 　　　　　　array[j + 1] = array[j]; 　　　　　 j--; 　　　　} 　　　　array[j + 1] = key; 　　} 　　console.timeEnd('插入排序耗时：'); 　　return array; } var arr=[3,44,38,5,47,15,36,26,27,2,46,4,19,50,48]; console.log(insertionSort(arr)); –>
