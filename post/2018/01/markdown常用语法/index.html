Markdown常用语法

2018年1月22日
| 常用命令

写博客，自然免不了接触Markdown,作为初学者，为了即用即查，我决定把相关命令记录下来，本文就是边学边写 此处声明：本文是 参考他人的文章&结合实践后，所写，另外，转载、复制随意，但请务必注明本文链接

标题：

写法:

# h1级标题
## h2级标题
### h3级标题
#### h4级标题
##### h5级标题
###### h6级标题

栗子： - #### 这是一个h4级标题

注意： - #与标题文字间要有空格

- 当-与#结合使用时，如果不想下一行受到干扰，可以使用Enter换行

分割线：

连续三个&三个以上的- 即可作出分割线

写法:

---

栗子：

注意：建议写分割线时，与上一行留出一空行

超链接：

写法1：

[连接名称](网址 , 标题)

[baidu](https://www.baidu.com "baidu")
[<i class="icon-refresh"></i> 点我刷新](/#)

栗子：

  • baidu
  • 点我刷新

写法2：

[链接名][链接代号]
[链接代号]:详细地址

[here][3]
[3]: http://www.baidu.com

栗子：

  • here

写法3：

直接展示链接：<http://www.baidu.com>

栗子：

  • 直接展示链接：http://www.baidu.com

键盘键

用法：

<kbd>Ctrl+[</kbd> and <kbd>Ctrl+]</kbd>

栗子：

  • Ctrl+[ and Ctrl+]

code格式：

反引号

用法：

Use the `printf()` function.
 ``There is a literal backtick (`) here.针对在代码区段内插入反引号的情况`

栗子：

  • Use the printf() function.

  • There is a literal backtick (`) here.针对在代码区段内插入反引号的情况

强调：

用法：

*斜体强调*
**粗体强调**

栗子：

  • 斜体强调
  • 粗体强调

图片：

用法：

![Alt text](../imgs/blogContentImgs/jhvh.jpg "这图我觉得扣得不错")
如果图片链接失效的话，会显示 Alt text

栗子：

使用 icon 图标文字：

用法：

<i class="fa fa-github"></i>

栗子：

段落：

以一个空行开始，以一个空行结束，中间的就是一个段落。

表格：

用法：

Item     | Value
-------- | ---
Computer | $1600
Phone    | $12
Pipe     | $1

栗子：

Item Value
Computer $1600
Phone $12
Pipe $1

无序列表：

使用 - 加一个空格

用法：

- 无需列表1
- 无序列表2
- 无序列表3

栗子：

  • 无需列表1
  • 无序列表2
  • 无序列表3

有序列表：

使用 数字 加一个英文句点 加一个空格

用法：

1. 有序列表
2. 有序列表
3. 有序列表
4. 有序列表
5. 有序列表

栗子：

  1. 有序列表
  2. 有序列表
  3. 有序列表
  4. 有序列表
  5. 有序列表

换行缩进形成代码区块：

用法：

这里先换行，然后缩进4个空格，之后的内容便可以原样显示了，适合用于显示代码内容。直到文本结束或最后一个存在缩进的行为止。

栗子：

这里先换行，然后缩进4个空格，之后的内容便可以原样显示了，适合用于显示代码内容。直到文本结束或最后一个存在缩进的行为止。

块引用

用法：

>给引用的文本开始位置都加一个 '>'，
>便可组成一个块引用。在块引用中，可以结合
>其他markdown元素一块使用，比如列表。
>**强调**
也可以只在第一行加大于号，其他位置不加。

>- 块引用里使用列表，需要和上面的内容隔开一个空行
>- 记得加空格哦。

栗子：

给引用的文本开始位置都加一个 ‘>‘， 便可组成一个块引用。在块引用中，可以结合 其他markdown元素一块使用，比如列表。 强调 也可以只在第一行加大于号，其他位置不加。

  • 块引用里使用列表，需要和上面的内容隔开一个空行
  • 记得加空格哦。
